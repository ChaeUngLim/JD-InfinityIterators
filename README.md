# JD-InfinityIterators (2조)

# 온라인 쇼핑몰 재고관리 및 상품구매 시스템
이 프로젝트는 온라인 쇼핑몰에서 간단한 재고관리 시스템과 상품구매 시스템을 구현하는 것을 목표로 했습니다.

## 기능

- 🛍️ 상품 등록, 수정, 삭제 기능
- 📊 재고 관리 기능
- 🛒 장바구니 기능
- 💳 상품 구매기능

### 프로젝트 구조
```
├── src/
│   ├── main/
│   │   ├── java/
│   │   │   └── com/
│   │   │       └── infinityiterators/ 
│   │   │           ├── Account/ - 계정 관련 클래스
│   │   │           ├── deps/ - 의존성 관련 클래스
│   │   │           ├── order/ - 주문 관련 클래스
│   │   │           ├── product/ - 상품 관련 클래스
│   │   │           ├── view/ - 뷰 관련 클래스
│   │   │           └── EntryPoint.java - 애플리케이션 실행 진입점
│   │   └── resources/ - 설정 파일 및 리소스
│   └── test/
│       └── java/ - 테스트 코드
├── .gitignore
├── build.gradle - Gradle 빌드 파일
├── gradlew
├── gradlew.bat - Gradle Wrapper 실행 파일
└── README.md
```

## 사용된 기술 스택

- Java 17
- IntelliJ
